# Infrastructure Datasets

## Purpose

Infrastructure datasets evaluate how easily a project can be constructed, connected, and operated.

Infrastructure readiness is often the difference between a viable and non-viable project.

---

# Business Questions

Infrastructure datasets help answer:

* Can the project be connected to the grid?
* Is suitable port infrastructure available?
* Are submarine cables nearby?
* Is construction logistics feasible?

---
