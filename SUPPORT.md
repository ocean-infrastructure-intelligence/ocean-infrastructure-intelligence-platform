# Support

Thank you for your interest in the Ocean Infrastructure Intelligence Platform (OIIP).

This document explains how to get help, ask questions, report problems, and participate in project discussions.

---

# Getting Help

If you need help using, understanding, or contributing to the project, please use the appropriate communication channel.

---

# Questions and Discussions

For general questions, ideas, and discussions, please use:

* GitHub Discussions
* Project documentation
* Existing Issues

Before creating a new discussion, please search existing topics to see if your question has already been answered.

Examples:

* How does the OTEC scoring model work?
* Which datasets are currently supported?
* How is the AI Datacenter Suitability Index calculated?
* How can I contribute oceanographic data?

---

# Reporting Bugs

If you discover a bug, please create a GitHub Issue.

When reporting a bug, include:

* Clear description of the problem
* Steps to reproduce
* Expected behavior
* Actual behavior
* Relevant logs or screenshots
* Environment information

Example:

```text
Operating System:
Python Version:
Database Version:
Project Version:
```

A well-documented bug report helps maintainers investigate and resolve issues more efficiently.

---

# Feature Requests

Feature requests are welcome.

Before creating a request:

* Search existing issues
* Search discussions
* Check the project roadmap

When proposing a feature, explain:

* The problem being solved
* The expected benefit
* Possible implementation ideas

Examples:

* New oceanographic data source
* Additional GIS capabilities
* New scoring methodology
* Infrastructure analysis tools
* AI agent functionality

# Research Discussions

OIIP is both a software and research initiative.

Research-related discussions may include:

* Marine engineering
